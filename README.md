# cleaning-website
Cleaning service website with booking functionality.

This project uses a mixed PHP structure.
Core backend logic is located in the `php/` directory.
Some PHP pages (admin and dashboard-related files) are located at the root level.
This structure reflects how the project was developed and tested locally.
When running the project locally, place the entire folder inside:
C:/xampp/htdocs/
